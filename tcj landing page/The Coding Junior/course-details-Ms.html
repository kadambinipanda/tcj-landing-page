Topics to be covered

Below mentioned are the topics that are covered in this course.

WORKING ON MS PAINT
Introduction and advantages, Saving the file, Adding Shapes, Filling colours

MS WORD
Introduction and Advantages, Components and commands, Short-cut keys, Working and saving a word file, Adding the image and borders, Using watermarks and adding page borders, Inserting header and footer

POWERPOINT PRESENTATION
Introduction and advantages, Adding images, Adding shapes, Adding Themes, Adding charts, Adding word art

MS EXCEL
Students can learn by using excel how to convert raw data into meaningful information. They will learn by doing exercises in Excel how to perform calculations faster. Introduction and advantages, Commands and alignment, Calculating salary using formulas, Conditional Formatting

Couse Objective

  • Indicating the names and functions of the Word .
  • Create, edit, save, and print documents including lists and tables.
  • Format text and uses of styles.
  • Adding header and footer in a document.
  • Add graphics to a document.
  • Manipulate documents using functions such as find and replace; cut, copy, replace and undo.
  • MS Excel gives an overview of basic excel formulas, filtering or sorting data.

Course Outcome

  1. Life-long learning: Recognize the need for, and have the preparation and ability to engage in independent and life-long learning in the broadest context of technological change.
  2. Understand the principles of MS Word, MA Excel and MS Powerpoint
  3. Recognize when to use each of the Microsoft Office programs to create professional and academic documents.
  4. Use Microsoft Office programs to create personal, academic and business documents.
